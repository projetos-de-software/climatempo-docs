# UC005 - Consultar Logs 

Este caso de uso refere-se a consulta de registros de envio de mensagens pelo dispositivo ao sistema pelo usuário. 

## Ator

Usuário.

## Pré-Condições

Ter um dispositivo cadastrado e funcional no sistema. 

## Fluxo Principal

|                                           Ator                                            |                  Sistema                   |
| :---------------------------------------------------------------------------------------: | :----------------------------------------: |
| O usuário informa ao sistema que deseja consultar os logs informando as datas disponíveis |                                            |
|                                                                                           | O sistema emite as informações solicitadas |

## Fluxo Alternativo

Não há. 

## Pós-Condições 

Não há. 

### Informações Adicionais

O sistema tem um limite de logs de 30 dias.
