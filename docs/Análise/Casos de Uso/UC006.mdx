# UC006 - Liberar Usuário

Este caso de uso refere-se ao administrador para a liberação de usuário para fazer uso do sistema. 

## Ator

Administrador

## Pré-Condições

O usuário precisará ter submetido um cadastro no sistema. 

## Fluxo Principal

|                         Ator                         |                         Sistema                          |
| :--------------------------------------------------: | :------------------------------------------------------: |
| O administrador escolhe o usuário e clica em liberar |                                                          |
|                                                      | O sistema emite a mensagem de que o usuário foi liberado |

## Fluxo Alternativo

Não há. 

## Pós-Condições 

O sistema envia um email ao usuário dizendo que ele está apto a utilizar o sistema.
