# UC004 - Monitorar Testes

Este caso de uso refere-se ao monitoramento do teste de um dispositivo pelo usuário no sistema. 

## Ator

Usuário

## Pré-Condições

O usuário precisa ter um dispositivo cadastrado no sistema. 

## Fluxo Principal

|                                 Ator                                 |                                Sistema                                 |
| :------------------------------------------------------------------: | :--------------------------------------------------------------------: |
| O ator informa ao sistema que deseja fazer um teste de monitoramento |                                                                        |
|                                                                      | O sistema mostra os registro de monitoramento em tempo real ao usuário |

## Fluxo Alternativo

Não há. 

## Pós-Condições 

Caso o teste seja bem sucedido, o sistema vai exibindo os resultados na tela.
