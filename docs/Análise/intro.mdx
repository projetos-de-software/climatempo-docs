# Análise

Este documento refere-se a parte de análise de software. 

## O Início - Ou de onde surgiu a ideia

Quando entrei no grupo da mecatrônica jovem, todas as pessoas que apresentam possuem um projeto próprio. A minha ideia era montar uma plataforma de telemetria colaborativa. A forma mais fácil de fazer isso é medindo a temperatura. Com um hardware barato, menos de 100 reais dá pra montar tudo, seria fácil fazer todos os processos. Aí começaram os problemas. Eu acreditei que fosse mais fácil de fazer que o normal...

## Descritivo Funcional 

Numa reunião da Mecatrônica Jovem no [canal do INCB](https://www.youtube.com/@InstitutoNCBNewtonCBraga) foi definido que seria feito uma plataforma colaborativa de telemetria IoT. O Marcos(eu) escolheu um hardware simples com MicroPython e um DHT11. Basicamente dois tipos de placa são particularmente interessantes: 

1. Raspberry Pi Pico W
2. Raspberry Pi Pico 2W 
3. ESP32 (tem um monte de variantes)

A versão dos Hardwares precisam de conectividade WiFI para enviar o sinal para internet. Isso permite que o site receba a informação da placa através da internet. A questão é que o site precisaria ser desenvolvido. 

Para algumas coisas no site eu vou falar nos documentos de [MVP](MVP) e [Recursos](Recursos).
