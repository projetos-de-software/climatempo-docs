# Recursos

Neste documento está descrito os recursos de software utilizados do lado do servidor. 

## Framework e Bibliotecas 

### Backend 

- [NextJS](https://nextjs.org/) - Versão 15.3.3 (atual) Framework Javascript onde a aplicação está sendo desenvolvida. 
- [PrismaORM](https://www.prisma.io/) - Mapeamento Objeto Relacional para Banco de dados
- [PM2](https://pm2.keymetrics.io/docs/usage/quick-start/) - Monitor de aplicações nodejs
- [ReactLeaf](https://react-leaflet.js.org/) - Recurso do OpenStreetMap que permite visualizar um ponto no mapa. 
- [NextAuth](https://next-auth.js.org/) - Biblioteca de Recursos para autenticação Oauth2 (Google, Github, etc)

#### Links de Implementação de Referência

[How JWT Works](https://www.descope.com/learn/post/jwt)

### Servidor 

- [MariaDB](https://mariadb.org/) - banco de dados do lado do servidor.
- [NGINX](https://nginx.org/) - Proxy Reverso para acesso aplicação.
