# Introdução

Este é o repositório de documentação do projeto [ClimaTempo](https://tempo.projetosdesoftware.com.br)

## História

A revista Mecatrônica Jovem lança uma edição a cada dois meses. Numa destas edições, sobre cidades inteligentes, eu sugeri que fosse feito uma estação meteorológica que enviasse informações para web. A partir daí eu comecei a trabalhar no projeto com uma Raspberry Pi Pico 2 W. Eu demorei mais que o normal porque tinha imaginado uma situação diferente e a edição mudou. 

A fim de conseguir concluir o primero projeto eu sugeri que o projeto tivesse um registro de acompanhamento.    

No dia 10 de julho de 2025 eu iniciei este site.
